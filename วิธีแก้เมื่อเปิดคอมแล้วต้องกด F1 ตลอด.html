วิธีแก้เมื่อเปิดคอมแล้วต้องกด F1 ตลอด เวลาในเครื่องก็ไม่ตรง เค้าซ่อมกันง่ายๆ แบบนี้เอง

วิธีแก้เมื่อเปิดคอมแล้วต้องกด F1 ตลอด สำหรับผู้ใช้งานคอมพิวเตอร์คงต้องเคยเจอเหตุการณ์เปิดเครื่องคอมพิวเตอร์แล้วตัวเครื่องเตือนให้กดปุ่ม F1 ตลอดถึงจะเข้าใช้งาน Windows ได้ พอเข้าไปแล้วเวลาในเครื่องก็ไม่ตรงกับเวลาปัจจุบันอีก เหนื่อยใจ...คนที่ตั้งเวลาคอมไม่เป็นก็จะเป็นปัญหาที่ใหญ่โตเอาการ คนที่ตั้งเป็นก็ต้องเสียเวลามาตั้งใหม่ทุกครั้งที่เปิดเครื่อง หากไม่ตั้งคุณก็จะมีปัญหากับ Facebook, Hotmail, Gmail หรือโปรแกรมต่างๆ ภายในเครื่อง เรามาวิเคราะห์กันก่อนว่าปัญหาเกิดจากอะไร

ปัญหาเปิดคอมแล้วต้องกด F1 นี้จะมาจาก 2 ส่วนหลักๆ

ภายในคอมพิวเตอร์ของเราจะมีแผงวงจรไฟฟ้าหรือที่ภาษาคอมพิวเตอร์เรียกว่า เมนบอร์ด (Mainboard) เจ้าเมนบอร์ดตัวนี้จะทำหน้าที่หลักๆ ของมันคือเชื่อมต่อกับอุปกรณ์อื่นๆ เช่น การ์ดจอ ฮาร์ดดิสก์ CPU Ram และอื่นๆ อีกมากมายให้ทำงานร่วมกันด้วยระบบสัญญาณนาฬิกา (สัญญาณนาฬิกาคืออะไรค่อยศึกษากันอีกที) โดยเมนบอร์ดนี้จะมีตัวชิป (Chip) ตัวหนึ่งเรียกว่าชิป Bios ทำหน้าที่บันทึกการตั้งค่าต่างๆ ของตัวเมนบอร์ดไว้
เช่น วันที่, การตั้งค่าอุปกรณ์ในการบูทเข้าระบบ และอื่นๆ แต่ชิป Bios ที่ว่านี้จำเป็นที่จะต้องมีไฟประมาณ 3V. เลี้ยงตัวชิปอยู่ตลอดเวลา หากไม่มีไฟเลี้ยงระบบก็จะถูก Reset กลับไปจุดเริ่มต้นอีกครั้ง เรียกง่ายๆ ว่าค่าโรงงาน หรือในบางกรณีชิป Bios ที่มีอายุการใช้งานมานานมากแล้วก็จะเกิดอาการไม่จดจำค่าได้เหมือนกัน มาถึงตรงนี้ก็พอจะรู้สาเหตุกันแล้วล่ะ ต่อไปมาดูวิธีการซ่อมในแต่ละส่วน

ส่วนแรก ตรวจเช็คว่ามีไฟ 3V. เลี้ยงวงจรอยู่หรือไม่ ฟังดูยากใช่มั้ยล่ะ ไม่ใช่ช่างไฟฟ้าจะให้มาเช็คไฟได้ไง ไม่ยากหรอกครับเจ้าไฟ 3V. ที่ว่านี้มันเป็นแค่แบตเตอรี่ก้อนเล็กๆ หรือภาษาชาวบ้านเราเรียกถ่านกระดุม, ถ่านนาฬิกา ใครนึกภาพไม่ออกดูภาพด้านล่างได้เลยครับ ในเมนบอร์ดของคอมตั้งโต๊ะ (Desktop) จะมีทุกรุ่น ในเมนบอร์ดของโน๊ตบุคจะมีบางรุ่นเท่านั้น (ส่วนมากมี) ถ่านตัวนี้หาซื้อได้ตามร้านซ่อมคอม หรือร้านขายอุปกรณ์อิเล็กทรอนิกส์ทั่วไป แพงสุดไม่เกิน 20 บาทต่อก้อน เกินนั้นถือว่าโดนหลอก

งัดออกมาจากเมนบอร์ดแล้วใช้มัลติมิเตอร์วัดครับ ตังย่านที่ DC 10V. อ่านค่าได้ต่ำกว่า 1.5V. เปลี่ยนทิ้งได้เลย แล้วถ้าเป็นคนบ้านๆ ไม่ใช่ช่างล่ะจะเอามัลติมิเตอร์มาจากไหน ก็ไม่ต้องหาครับเปลี่ยนถ่านก้อนใหม่เข้าไปเลยลงทุน 20 บาท

วิธีการตรวจเช็คว่าใช้งานได้หรือปล่าวก็ให้เปิดเครื่องตามปกติ ในครั้งแรกจะยังขึ้น F1 อยู่ (เพราะเราถอดถ่าน 3V. ออกมันเลย Reset) เข้าหน้า Bios ตั้งวันที่เวลาให้ตรงแล้วเข้า Windows ได้เลย ถ้าตั้งค่าใน Bios ไม่เป็นให้กด F1 เข้า Windows ไปก่อนครับเราสามารถตั้งวันที่เวลาใน Windows ได้เหมือนกัน โดยคลิกเข้าไปดูขั้นตอนจากบทความนี้ การตั้งค่าวันที่และเวลาใน Windows

ขั้นตอนต่อไปให้เรา Showdown ปิดเครื่องตามปกติและเมื่อเครื่องดับสนิทแล้วให้เราถอดปลั๊กไปออกไม่ให้มีไฟไหลผ่านตัวเครื่อง ทิ้งไว้ประมาณ 10 นาทีแล้วเปิดเครื่องอีกครั้ง ถ้าสามารถเข้า Windows ได้ตามปกติโดยไม่ขึ้น F1 ก็จบปัญหา แต่ถ้ายังขึ้นอยู่แสดงว่ากำลังเจอกับปัญหาที่หนักกว่า

ส่วนที่สอง หากแก้ปัญหาในส่วนแรกแล้วไม่หายอาจเป็นไปได้ว่าชิป Bios เสียซึ่งอาจจะมาจากการช๊อตของเมนบอร์ด, กระแสไฟเกิน, หรืออายุการใช้งาน เจ้าชิป Bios ไม่มีขายครับต้องถอดมาจากเมนบอร์ดรุ่นเดียวกันเท่านั้นและขั้นตอนยุ่งยากมาก ใช้อุปกรณ์ที่ราคาแพงกว่าเมนบอร์ดใหม่เสียอีก เพราะฉะนั้นการเปลี่ยนเมนบอร์ดใหม่น่าจะเป็นทางเลือกที่ดีที่สุดในการแก้ปัญหาของส่วนที่สองนี้
